# ericlifka.com

Uses [ES6 Template](https://github.com/ericlifka/es6-template)
Supplies builds for [ericlifka.github.io](https://github.com/ericlifka/ericlifka.github.io)

## Dependencies

Dev dependencies installed via npm
```
$> npm install
```

App dependencies installed via bower
```
$> bower install
```

## Develop

Powered by [broccoli](https://github.com/broccolijs/broccoli)
```
$> npm install -g broccoli
$> broccoli serve
```
